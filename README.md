# My Thoughts

A macOS app for capturing and organizing your thoughts.

## Features

- Clean, minimal interface
- Built with SwiftUI
- Native macOS experience

## Requirements

- macOS 15.5 or later
- Xcode 16.4 or later

## Building

1. Clone the repository
2. Open `my-thoughts.xcodeproj` in Xcode
3. Build and run the project
